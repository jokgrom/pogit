dir или ls показывает в какой мы папке и что содержит
cd - переход по папкам
git init делает инициализацию папки(создаёт скрытую папку .git)
git status
git add <имя файла и расширение> - добавляет для слешки файла
git add . добавляет все файлы
git rm --cached <имя файла> удаляет из слежки(больше не следит за изменением файла)
git commit -m "название комита" делаем точку сохранения приложения
